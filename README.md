# Mémo AIS

Petit projet MkDocs prêt à l'emploi pour centraliser des notes de formation AIS.

## Installation

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
```

## Générer le site

```bash
mkdocs build
```

Le site généré se trouve dans le dossier `site/`.
