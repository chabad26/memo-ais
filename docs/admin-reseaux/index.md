# Administration des réseaux - Fondamentaux

Les notions  vue dans ce module seront les suivants :

- Décrire le modèle TCP/IP, nommer les protocoles par couche et identifier leur rôle dans un échange réseau.
- Concevoir un plan d'adressage IPv4 découpé en sous-réseaux avec la notation CIDR (Classless Inter-Domain Routing).
-Configurer des équipements réseau (switches, routeurs) sur simulateur et sur matériel physique.
- Mettre en œuvre les services fondamentaux d'une infrastructure réseau : DHCP (Dynamic Host Configuration Protocol), NAT (Network Address Translation), DNS (Domain Name System).
- Analyser un trafic réseau avec Wireshark et en extraire des informations pertinentes pour le diagnostic.
- Diagnostiquer une panne réseau avec une démarche méthodique par couches OSI (Open Systems Interconnection).
- Documenter une configuration réseau au standard professionnel (en-tête, commentaires, réimportable).
- Écrire un script Bash d'automatisation documenté et réexécutable.
