# Services réseau

📌 Au programme de cette itération

Découvrir et configurer les trois services réseau qui rendent une infrastructure réellement exploitable : DHCP (attribution automatique des IPs avec le processus DORA), NAT/PAT (accès Internet partagé), DNS interne (BIND9 sur la zone alpesnet.local). En fin de journée, Autonomie 2 de 3h pour finaliser et documenter l'infrastructure complète AlpesNet — livrable RNCP L3.
