# Enjeux clés d'un SI

Cette partie regroupe les feuilles liées aux enjeux et aux risques d'un SI :

- disponibilité ;
- sécurité ;
- évolutivité ;
- intégration ;
- conformité ;
- RTO / RPO ;
- identification et priorisation des risques.

L'objectif est de relier les notions d'architecture à des situations concrètes.
