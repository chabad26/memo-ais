# Système d’info & archi SI

Un **système d'information** regroupe les personnes, les outils, les données, les procédures et les infrastructures qui permettent à une organisation de fonctionner.

L'**architecture SI** sert à organiser ces éléments pour que les applications, les serveurs, les réseaux et les services travaillent ensemble de manière cohérente.

Cette partie permet de comprendre comment un SI est structuré, comment les flux circulent, quels composants sont essentiels et pourquoi la sécurité doit être pensée dès la conception.

## À retenir

- Le SI soutient les activités de l'entreprise.
- L'architecture SI décrit l'organisation technique et fonctionnelle du système.
- Une bonne architecture facilite l'administration, la supervision, l'évolution et la sécurité.
