# Sécurité d'un SI

Ici, l'objectif est de voir les systèmes de sécurité possibles :

- VPN ;
- pare-feu ;
- segmentation ;
- supervision ;
- continuité d'activité.

On cherche surtout à comprendre comment réduire l'impact d'un incident : bloquer une propagation, limiter une fuite, éviter une indisponibilité massive et préparer la reprise.
