# Le système d'information

Cette partie regroupe les feuilles liées à la cartographie du SI :

- cas concret du CHU de Rouen ;
- vue globale ;
- diagramme réseau ;
- cartographie des utilisateurs ;
- architectures applicatives ;
- dépendances ;
- diagrammes finaux.

L'objectif est de comprendre comment un SI est structuré et comment ses composants dépendent les uns des autres.
