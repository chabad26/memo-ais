# Pense-bête

Le pense-bête devient un **glossaire opérationnel**.

Il ne remplace pas les pages de cours : il sert à retrouver rapidement les termes, les commandes et les manipulations déjà faites. Chaque fiche correspond à une itération ou à un bloc de cours, puis renvoie vers les fichiers Markdown associés.

## Règle d'organisation

- Une itération = une fiche.
- Une fiche = définitions + manipulations + liens vers les docs.
- Les pages de cours restent dans leurs dossiers d'origine.
- Le pense-bête évite les longs développements : il garde les mots-clés et les gestes à retenir.
