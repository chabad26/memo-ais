# Glossaire Réseaux - Itération 1

## Sujet

Fondations TCP/IP, adressage IPv4, subnetting, Wireshark et schémas L1/L2/L3.

## Termes à retenir

| Terme | Définition courte |
| --- | --- |
| TCP/IP | Ensemble de protocoles permettant la communication réseau. |
| IPv4 | Adresse logique sur 32 bits, souvent notée en décimal pointé. |
| Masque | Valeur qui sépare partie réseau et partie hôte. |
| Sous-réseau | Découpage logique d'un réseau IP. |
| Passerelle | Equipement utilisé pour sortir du réseau local. |
| DNS | Service de résolution de noms en adresses IP. |
| Wireshark | Outil de capture et d'analyse de paquets. |
| Couche L1 | Support physique : câble, radio, signal. |
| Couche L2 | Liaison locale : Ethernet, MAC, switch. |
| Couche L3 | Routage IP entre réseaux. |

## Manipulations faites

| Manipulation | Commandes ou actions |
| --- | --- |
| Calculer un réseau | Adresse réseau, broadcast, plage utilisable, CIDR. |
| Capturer du trafic | Wireshark, filtres ICMP/DNS/ARP. |
| Vérifier une interface | `ip addr`, `ip link`, `ip route`. |
| Tester la connectivité | `ping`, `traceroute`, `dig`. |
| Produire un schéma | Identifier liens physiques, VLANs, réseaux IP et passerelles. |
