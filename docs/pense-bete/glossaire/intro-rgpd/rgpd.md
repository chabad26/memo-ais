# Glossaire - RGPD

## Termes à retenir

| Terme | Définition courte |
| --- | --- |
| RGPD | Règlement européen encadrant le traitement des données personnelles. |
| Donnée personnelle | Information permettant d'identifier directement ou indirectement une personne. |
| Traitement | Toute opération sur des données : collecte, stockage, consultation, modification, suppression. |
| Responsable de traitement | Entité qui décide pourquoi et comment les données sont traitées. |
| Sous-traitant | Entité qui traite les données pour le compte du responsable. |
| DPO | Délégué à la protection des données, référent conformité et conseil. |
| Base légale | Justification juridique du traitement : consentement, contrat, obligation légale, intérêt légitime, etc. |
| Minimisation | Ne collecter que les données nécessaires à l'objectif. |
| Durée de conservation | Temps pendant lequel les données peuvent être gardées. |
| Violation de données | Incident portant atteinte à la confidentialité, l'intégrité ou la disponibilité de données personnelles. |

## Manipulations faites

| Manipulation | Action |
| --- | --- |
| Identifier les acteurs | Distinguer responsable, sous-traitant, personne concernée et DPO. |
| Analyser un traitement | Relever finalité, données, base légale, durée, destinataires et sécurité. |
| Appliquer les règles d'or | Vérifier finalité, minimisation, transparence, sécurité et droits des personnes. |
| Lire des fiches CNIL | Utiliser les PDF comme support de synthèse. |
