# Glossaire SI - Itération 1

## Sujet

Le système d'information, ses composants, ses utilisateurs, ses flux et ses dépendances.

## Termes à retenir

| Terme | Définition courte |
| --- | --- |
| Système d'information | Ensemble organisé de personnes, processus, données, applications et infrastructures. |
| Application métier | Logiciel utilisé pour réaliser une activité de l'organisation. |
| Infrastructure | Socle technique : serveurs, réseau, stockage, postes, services. |
| Flux | Circulation d'une donnée entre deux composants ou acteurs. |
| Donnée critique | Donnée dont la perte ou l'altération perturbe fortement l'activité. |
| Dépendance applicative | Relation entre applications ou services nécessaires au fonctionnement. |
| Cartographie | Représentation structurée du SI pour comprendre composants, flux et risques. |
| Chemin d'attaque | Suite d'étapes possibles permettant à un attaquant d'atteindre une cible. |

## Manipulations faites

| Manipulation | Objectif |
| --- | --- |
| Décrire un SI réel | Identifier acteurs, applications, données et infrastructures. |
| Représenter les flux | Montrer qui communique avec quoi et pourquoi. |
| Distinguer vues | Séparer vue réseau, applicative, utilisateurs et risques. |
| Analyser les dépendances | Comprendre ce qui tombe si un service ou une application tombe. |
